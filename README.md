# Trae Me

`Trae Me` is a custom dark theme pack for [Zed](https://zed.dev), based on the Trae IDE look but adjusted for a cleaner Zed experience.

This extension includes two themes:

- `Trae Me` — the standard dark variant
- `Trae Me Glass` — a blurred / translucent variant


## Main Palette

The theme uses a compact dark palette built around these main colors:

- `#181818` — primary UI surface
- `#1b1b1b` — active editor / active tab / terminal surface
- `#202122` — active line and elevated dark surface
- `#2a2b2c` — borders and guide lines
- `#333536` — active guide / stronger border contrast
- `#bfbfbf` — primary text
- `#8c8c8c` — muted text and icons
- `#72c892` — added / success
- `#e5ba7d` — modified / warning
- `#f28772` — deleted / error
- `#3a94bc` — blue accent used across interactive states
